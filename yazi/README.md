# Yazi

Terminal file manager with minimal configuration and Dracula color flavor.

## Dependencies

- `yazi` -- install via `brew install yazi`

## Configuration

Settings in `yazi.toml`:

| Setting | Value |
|---------|-------|
| Show hidden files | `true` |
| Sort directories first | `true` |
| Sort by | `natural` |
| Line mode | `mtime` (modification time) |
| Show symlinks | `true` |
| Flavor | `dracula` |

## Theme

The Dracula flavor is installed at `flavors/dracula.yazi/flavor.toml` and applies Dracula colors to the entire Yazi UI.
