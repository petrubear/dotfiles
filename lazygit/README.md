# lazygit

Terminal UI for Git with a custom Dracula color theme.

## Dependencies

- `lazygit` -- install via `brew install lazygit`

## Configuration

### Dracula Theme

Custom color palette applied to the lazygit UI:

| Element | Color | Description |
|---------|-------|-------------|
| Active border | `#FF79C6` (pink, bold) | Currently focused panel border |
| Inactive border | `#BD93F9` (purple) | Unfocused panel borders |
| Searching active border | `#8BE9FD` (cyan, bold) | Panel border during search |
| Options text | `#6272A4` (comment) | Help/options text |
| Selected line background | `#6272A4` (comment) | Highlighted line background |
| Cherry-picked commit foreground | `#6272A4` (comment) | Cherry-pick marker text |
| Cherry-picked commit background | `#8BE9FD` (cyan) | Cherry-pick marker background |
| Marked base commit foreground | `#8BE9FD` (cyan) | Rebase base marker text |
| Marked base commit background | `#F1FA8C` (yellow) | Rebase base marker background |
| Unstaged changes | `#FF5555` (red) | Unstaged file indicator |
| Default foreground | `#F8F8F2` (foreground) | Default text color |
